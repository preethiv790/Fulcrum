# Fulcrum

This is a simple two page application that built on top of Google Map address API. 

The following set of libraries are used in this project

- [x] RxJava
- [x] Retrofit
- [x] Dagger 2
- [x] GSON

## Design Pattern
 MVVM 

## Navigation Component
Jetpack Navigation
